Про наш навчальний проект

Цей веб-сайт створено в рамках вивчення веб-технологій та каскадних таблиць стилів. Метою проекту є демонстрація можливостей використання зовнішніх CSS-файлів для створення єдиного стилю оформлення для всіх сторінок сайту. Використання зовнішніх таблиць стилів значно спрощує підтримку та оновлення дизайну веб-сайту.

Наша команда та навчальний процес

Ми - студенти технічного університету, які вивчають сучасні веб-технології та їх практичне застосування. У процесі навчання ми опановуємо HTML, CSS, JavaScript та інші технології, необхідні для створення професійних веб-сайтів. Наші лабораторні роботи спрямовані на поступове освоєння всіх аспектів веб-розробки.

Кожна лабораторна робота присвячена певному аспекту веб-технологій. Ми вивчили створення структури HTML-документів, форматування тексту, роботу з таблицями, списками, формами, зображеннями та гіперпосиланнями. Поточна лабораторна робота присвячена вивченню каскадних таблиць стилів та їх практичному застосуванню.

Що ми вивчили?

В рамках курсу веб-технологій ми освоїли:

  1. Основи HTML - структура документів, семантичні теги
  2. Робота з текстом - форматування, стилізація
  3. Таблиці - створення та оформлення табличних даних
  4. Списки - упорядковані, неупорядковані, списки визначень
  5. Гіперпосилання - створення навігації між сторінками
  6. Зображення та мультимедіа - вставка та оптимізація
  7. Картировання зображень - інтерактивні графічні карти
  8. Форми - створення елементів введення даних
  9. CSS - внутрішні та зовнішні таблиці стилів
  10. Стильові класи - багаторазове використання стилів

Переваги використання CSS

Каскадні таблиці стилів надають численні переваги при розробці веб-сайтів. По-перше, вони дозволяють відокремити контент від презентації, що робить код чистішим та легшим для підтримки. По-друге, один CSS-файл може використовуватися для багатьох HTML-сторінок, забезпечуючи єдність дизайну всього сайту.

CSS також значно розширює можливості оформлення порівняно з чистим HTML. Можна легко змінювати кольори, шрифти, відступи, розміщення елементів, додавати тіні, градієнти, анімації та багато іншого. Сучасний CSS3 підтримує створення адаптивного дизайну, який автоматично підлаштовується під різні розміри екранів.
