Корисні ресурси Інтернету

Google
Google − один із найпопулярніших в Україні та світі пошукових серверів, що надає швидкий доступ до інформації.

Gmail
Gmail − безкоштовна поштова служба від Google з великим обсягом пам'яті та захистом від спаму.

YouTube
YouTube − найбільший відеохостинг у світі для перегляду та публікації відеоконтенту.

Вікіпедія
Вікіпедія − вільна енциклопедія, яку може редагувати кожен, з мільйонами статей різними мовами.

GitHub
GitHub − найбільша платформа для розробників для зберігання коду та спільної роботи над проектами.

Stack Overflow
Stack Overflow − найбільше співтовариство програмістів для пошуку відповідей на технічні питання.

LinkedIn
LinkedIn − професійна соціальна мережа для встановлення бізнес-контактів та пошуку роботи.

Google Translate
Google Translate − сервіс машинного перекладу текстів та веб-сторінок більш ніж 100 мовами світу.
